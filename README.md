# Banashree Resort Website

A modern and responsive website developed for **Banashree Resort**, Joypur, West Bengal. The website is designed to showcase the resort's facilities, rooms, restaurant, gallery, and contact information with a clean user interface and responsive design.

---

## Live Demo

GitHub Pages:
https://kartickkole.github.io/banashree-resort-website/

---

## Features

- Responsive design for desktop, tablet and mobile
- Hero image slider
- About Resort section
- Room information
- Restaurant section
- Facilities section
- Image gallery
- Customer reviews
- Nearby attractions
- Google Maps integration
- Contact information
- WhatsApp and Call buttons
- Smooth scrolling
- Mobile navigation
- SEO friendly structure

## Installation

Clone the repository

```bash
git clone https://github.com/KartickKole/banashree-resort-website.git
```

Go to project folder

```bash
cd banashree-resort-website
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
